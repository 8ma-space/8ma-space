DROP YOUR PHOTOS IN THIS FOLDER
================================

The website is wired so photos appear automatically the moment
you save them here with the right filename. No HTML editing needed.

Just download from your Instagram (https://www.instagram.com/8ma.space/)
and rename each file to one of the names below.

Required filenames:

  hero.jpg              -> Big photo on the home page (right of headline)
  about.jpg             -> Smaller photo on the home page "About" block
  about-portrait.jpg    -> Main portrait on the About page
  about-secondary.jpg   -> Candid/secondary photo on the About page
  challenge.jpg         -> Photo on the Weekly Challenge page
  program.jpg           -> Photo on the Program page

How to download from Instagram:
  1. Open https://www.instagram.com/8ma.space/ in your browser.
  2. Click the photo you want to use.
  3. Right-click the photo and choose "Save image as..."
  4. Save it to this folder, naming it from the list above (e.g. hero.jpg).
  5. Reload the website page in your browser. The photo will appear.

Tips:
  - Square or portrait orientation works best.
  - Keep each file under 500 KB so the site stays fast.
    Compress with https://squoosh.app or https://tinypng.com if needed.
  - .png and .webp also work — but you'd need to update the filename
    in the HTML to match. Stick with .jpg for the easiest path.
